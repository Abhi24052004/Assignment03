# Assignment03



## p01.py — Factorial (recursive)
- What it does: Computes the factorial of a positive integer using a recursive helper.
- Input: One integer (n) read from stdin
- Output: Prints Factorial of <n> is: <result> to stdout.
- Example run:
    python p01.py
    Enter a number: 5
    Factorial of 5 is: 120


## p02.py — Basic math functions
- What it does: Reads an integer, then prints its square root, natural logarithm, and sine from math module.
- Input: One integer (n) read from stdin.
- Output: Three lines labeled `Square Root`, `Logarithm` (natural log), and `Sine`.
- Example run:
    python p02.py
    Enter a number: 9
    Square Root: 3.0
    Logarithm: 2.1972245773362196
    Sine: 0.4121184852417566
